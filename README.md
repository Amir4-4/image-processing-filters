
#  Image Processing Filters Project

##Project Overview
This project is an Image Processing application developed using Python. It applies various filters and transformations on images to demonstrate fundamental concepts of Digital Image Processing.

The project was implemented using Google Colab and Python libraries.

---

## Features
- Convert image to Grayscale
- Apply Blur filter
- Edge Detection (Sobel / Canny)
- Image Sharpening
- Brightness and Contrast adjustment
- Display original vs processed images

---

## Technologies Used
- Python
- NumPy
- OpenCV
- Matplotlib
- Google Colab

---


##  Filters Implemented
- Grayscale Conversion
- Gaussian Blur
- Edge Detection
- Image Sharpening
- Intensity Adjustment
